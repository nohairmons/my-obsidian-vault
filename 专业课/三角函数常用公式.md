### 一、 倍角公式 (Double-Angle Formulas)

通过和角公式推导而来，是解方程和化简表达式的核心。

- **正弦倍角：**
    
    $$\sin(2\theta) = 2\sin\theta\cos\theta$$
    
- **余弦倍角：**
    
    $$\cos(2\theta) = \cos^2\theta - \sin^2\theta = 2\cos^2\theta - 1 = 1 - 2\sin^2\theta$$
    
- **正切倍角：**
    
    $$\tan(2\theta) = \frac{2\tan\theta}{1 - \tan^2\theta}$$
    

### 二、 和差化积公式 (Sum-to-Product Formulas)

用于将和或差的形式转化为乘积形式，便于后续约分或化简。

- $\sin A + \sin B = 2\sin\left(\frac{A+B}{2}\right)\cos\left(\frac{A-B}{2}\right)$
    
- $\sin A - \sin B = 2\cos\left(\frac{A+B}{2}\right)\sin\left(\frac{A-B}{2}\right)$
    
- $\cos A + \cos B = 2\cos\left(\frac{A+B}{2}\right)\cos\left(\frac{A-B}{2}\right)$
    
- $\cos A - \cos B = -2\sin\left(\frac{A+B}{2}\right)\sin\left(\frac{A-B}{2}\right)$
    

### 三、 积化和差公式 (Product-to-Sum Formulas)

常用于将乘积形式转化为和差，便于积分运算或处理信号分析中的频率分量。

- $\sin A\cos B = \frac{1}{2}[\sin(A+B) + \sin(A-B)]$
    
- $\cos A\sin B = \frac{1}{2}[\sin(A+B) - \sin(A-B)]$
    
- $\cos A\cos B = \frac{1}{2}[\cos(A+B) + \cos(A-B)]$
    
- $\sin A\sin B = -\frac{1}{2}[\cos(A+B) - \cos(A-B)]$
    

### 四、 其他常用基础公式速查

|**名称**|**公式**|
|---|---|
|**同角基本关系**|$\sin^2\theta + \cos^2\theta = 1$|
|**半角公式**|$\sin\frac{\theta}{2} = \pm\sqrt{\frac{1-\cos\theta}{2}}, \cos\frac{\theta}{2} = \pm\sqrt{\frac{1+\cos\theta}{2}}$|
|**辅助角公式**|$a\sin\theta + b\cos\theta = \sqrt{a^2+b^2}\sin(\theta+\varphi)$ (其中 $\tan\varphi = \frac{b}{a}$)|
